Expected folder structure:
December 1/ → Doe, John - 1234567890/ → [application.pdf, evaluation.pdf, interview.mp4, …]
The candidate's name and ID are read from the folder name. All files inside are attached to their profile.
Auto-batching rules:
  • Each batch holds a maximum of 10 candidates.
  • If the most recent batch for this dataset has open slots, new candidates fill it first and the batch name is updated.
  • Batch names come from the uploaded folder name (e.g. "December 1").
  • If overflow is needed: "December 1 (1)", "December 1 (2)", etc.
  • Multiple folders combine names: "December 1 - December 2".
